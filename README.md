# Community-Micro-Alert-Network-like-Waze-for-local-issues-
A web/mobile app to allow residents to report and see small local issues like broken streetlights, lost pets, flooding, or suspicious activity, with real-time updates and a map-based interface.

### Problem Statement

Local small-scale problems are often ignored or reported in fragmented ways (WhatsApp, Facebook groups). This app provides a centralized, real-time, community-driven platform.

### Features (planned)*
* Submit issues with text, photo, category, and location
* View alerts on a real-time interactive map
* Notifications for nearby alerts
* Optional: Upvote, comment, AI categorization, gamification

### Tech Stack (planned)
* Frontend: React.js or Flutter
* Backend: Firebase or Node.js + Express
* Database: Firestore (real-time)
* Maps: Google Maps API or Mapbox
* Notifications: Firebase Cloud Messaging
